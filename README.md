
# element-ui 组件库

### 快速开始

#### 1.安装组件库
```bash
    npm i element-he
```
#### 2.引用组件库
```javascript
// 全部引入
   import 'element-he/css/index.css'
   import MUI from 'element-he'
   Vue.use(element-he)
// 按需引入
    import 'element-he/css/index.css'
    import {table} from 'element-he'
    Vue.use(table)      
```
